### 1、指针接收&值接收
指针接收：实现接口的方法中接收者是指针类型    
值接收：实现接口的方法中接收者是值类型   
```
package main

import "fmt"

type Animal interface {
	eat()
}
type Dog struct {
	weight int
}
func (d *Dog) eat() {
	fmt.Println("Dog is eating")
}
type Cat struct {
	weight int
}
func (c Cat) eat() {
	fmt.Println("Cat is eating")
}
func eat(o Animal) {
	o.eat()
}
func main() {
	dog := Dog{5}
	eat(&dog)

	cat := Cat{6}
	#eat(&cat)
	eat(cat)
}
```
上述代码中，Dog是指针接受，Cat是值接收，在使用时一般是定义一个多态还是如：
```
func eat(o Animal) {
	o.eat()
}
```
在调用时，**指针接收类型的实现只能传入指针类型，但是值类型接收既可以传入指针类型也可以传入值类型**
